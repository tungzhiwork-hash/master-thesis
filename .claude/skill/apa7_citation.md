# Chuẩn Trích Dẫn APA 7 — Tham Chiếu Nhanh

## Bài báo tạp chí (dạng phổ biến nhất)
```
Tác giả, T. T., & Tác giả 2, T. (Năm). Tên bài báo. Tên Tạp Chí, số(tập), trang. https://doi.org/xxxxx
```

**Ví dụ:**
```
Tran, A. T., Nguyen, B. H., & Le, C. D. (2025). Comparative analysis of boosting algorithms
for predicting personal default. Cogent Economics & Finance, 13(1), Article 2456789.
https://doi.org/10.1080/xxxxxxx
```

## Báo cáo tổ chức / NHNN
```
Tên tổ chức. (Năm). Tên báo cáo. Nhà xuất bản hoặc URL.
```

**Ví dụ:**
```
Ngân hàng Nhà nước Việt Nam. (2024). Báo cáo thường niên 2023.
https://www.sbv.gov.vn/xxxxxxx
```

## Dataset công khai
```
Tác giả/Tổ chức. (Năm). Tên dataset [Dataset]. Nền tảng. https://doi.org/xxxxx
```

**Ví dụ:**
```
LendingClub. (2023). LendingClub loan data [Dataset]. Kaggle.
https://www.kaggle.com/datasets/xxxxxxx
```

## Trích dẫn nội tuyến (in-text)

| Trường hợp | Format |
|-----------|--------|
| Một tác giả | `(Nguyen, 2024)` |
| Hai tác giả | `(Tran & Le, 2025)` |
| Ba tác giả trở lên | `(Markov et al., 2022)` |
| Trích dẫn trực tiếp < 40 từ | `(Petter et al., 2022, tr. 5)` |
| Nhiều nguồn cùng lúc | `(Chang et al., 2022; Gramegna & Giudici, 2021)` — sắp xếp theo alphabet |

## Quy tắc nguồn được phép dùng
- ✅ Ưu tiên: tạp chí SSCI / SCIE / Scopus; Q1/Q2
- ✅ Tạp chí trong nước: chỉ dùng loại được Hội đồng chức danh GS Nhà nước chấm ≥ 0.75 điểm
- ❌ Tránh: Beall List (tạp chí predatory)
- ❌ Không dùng: tài liệu chưa peer-reviewed (blog, wiki, báo phổ thông) làm nguồn học thuật chính
